# IMO 2020 A3

Find the smallest possible value of $\displaystyle \frac{a}{b} + \frac{b}{c} + \frac{c}{d} + \frac{d}{a}$ across all positive real numbers $a$, $b$, $c$, and $d$ satisfying $(a + c)(b + d) = ac + bd$.



# Answer

$8$.
