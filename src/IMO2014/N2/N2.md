# IMO 2014 N2 (Integer Version)

Determine all pairs $(x, y)$ of integers such that
$$ 7x^2 - 13xy + 7y^2 = (|x - y| + 1)^3. $$



# Answer

$(x, y) = (m^3 + m^2 - 2m - 1, m^3 + 2m^2 - m - 1)$ and $(x, y) = (m^3 + 2m^2 - m - 1, m^3 + m^2 - 2m - 1)$ for some $m \in \Z$.
